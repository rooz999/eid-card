# مولّد بطاقة تهنئة عيد الفطر بالاسم
#Eid Greeting Card Generator

## العربية

صفحة ويب بسيطة تتيح للمستخدم كتابة اسمه داخل بطاقة تهنئة بعيد الفطر، ثم معاينة البطاقة وتحميلها كصورة جاهزة للمشاركة.

تم تصميم البطاقة خصيصًا لزوار الروضة الشريفة في المسجد النبوي الشريف.

### المميزات

- كتابة الاسم داخل البطاقة
- معاينة فورية للبطاقة
- تحميل البطاقة كصورة
- يعمل مباشرة في المتصفح بدون خادم
- مناسب للجوال والكمبيوتر

### طريقة الاستخدام

1. اكتب الاسم في الحقل المخصص
2. اضغط **معاينة**
3. اضغط **تحميل الصورة** لتنزيل البطاقة

### التقنيات المستخدمة

- HTML
- CSS
- JavaScript
- Canvas API

## English

A simple web page that allows users to generate a personalized Eid greeting card by entering their name, previewing the card, and downloading it as an image.

The card is designed for visitors of **Al-Rawdah Al-Sharifah** in Al-Masjid An-Nabawi.

### Features

- Add name to the greeting card
- Live preview
- Download the final card as an image
- Works directly in the browser
- Mobile friendly

### How to Use

1. Enter the name in the input field
2. Click **Preview**
3. Click **Download Image** to save the card

### Technologies

- HTML
- CSS
- JavaScript
- Canvas API
